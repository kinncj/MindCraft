# Product vision

MindCraft is a creative block-building toy for young kids, playable in any modern browser.

A 6-year-old calls Minecraft "MindCraft". Real Minecraft is a bit much at that age:
monsters, hunger, death screens, servers, chat. This game keeps the part that kid
actually loves — putting colorful blocks on other blocks — and drops everything else.

## What it is

- A small, bright 3D world made of blocks
- Creative mode only: place, remove, decorate, done
- A Magic Delivery Box (a cute cardboard box) that stores blocks, because the same kid
  also loves delivery boxes
- Local-first: the world saves in the browser, exports to a file, imports back

## What it will never be

- Online. No multiplayer, no chat, no sharing platform, no accounts.
- Scary. No enemies, no damage, no dying, no darkness, no jump scares.
- Monetized. No ads, no store, no unlockables, no tracking.
- A backend service. It is a static page. GitHub Pages can host it forever for free.

## Who it's for

A ~6-year-old at a keyboard or trackpad, possibly with a parent nearby. Reading ability
is limited: buttons carry emoji and short labels, and nothing important hides behind text.

## How we know it's good

The kid opens it, builds something, closes the laptop, opens it the next day, and the
build is still there. No adult intervention needed between those steps.
